h1. Diibar

h2. ABOUT

*In Development, almost!!*
Diibar is a Diigo client in menubar for Mac OS X.

h2. FEATURE

* Sync diibar's bookmarks
* Show your bookmarks in menu-bar
* Open bookmark with default browser
